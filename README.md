#  Dokumentasjon – Restaurantnettsted med HTML og CSS

Dette prosjektet består av et tilgjengelig nettsted for en restaurant, utviklet med kun HTML og CSS.

## Struktur

Nettstedet inneholder tre hovedsider:

- **Forside (`index.html`)**
  - Viser restaurantens navn
  - Bildegalleri med restaurantmeny
  - To navigasjonsknapper med bilder som leder til bestilling og tilgjengelighet

- **Bestillingsside (`bord.html`)**
  - Skjema der brukeren kan reservere bord
  - Kontaktinformasjon

- **Tilgjengelighetsside (`tilgjengelighet.html`)**
  - Informasjon om fysiske tilpasninger, sansetilpasning og matallergier
  - Kontaktinformasjon for spesielle behov

## Tilgjengelighet og Universell Utforming

Alle sidene er bygget med semantiske HTML-elementer:

- `<header>`, `<main>`, `<section>`, `<footer>`, `<form>`
- Bilder inkluderer `alt`-tekst for skjermlesere
- Skjemaet er strukturert med etiketter og felt som støtter tastaturnavigasjon og skjermlesere

CSS-stilen er kontrastvennlig, med fokus på:

- Lesbarhet
- Tydelig struktur

## Navigasjon

- Intuitiv navigasjon
- Logoen fungerer som en lenke tilbake til forsiden

## Videre Utvikling

Prosjektet er laget med tanke på universell utforming, og gir et godt utgangspunkt for:

- Mer funksjonalitet
- Responsiv design
